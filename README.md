# Tip-Calculator

A simple app built from scratch, that calculate tip for each person in a party based on bill amount and added tip.
You can see deployed app [here](https://mladen1506.github.io/tip-calculator/)

## Technologies Used

 * HTML
 * CSS
 * JavaScript
